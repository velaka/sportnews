How to install the project ?

1. First need localhost on your computer.
2. Composer.
3. Set up your Database in localhost.
4. Open the cmd.
5. With cd/ command go to project location.
6. Use this command to install composer - composer install.
7. Use php artisan migrate to create all tables in database.
8. Generate APP_KEY with this command - php artisan key:generate.
9. Config .env file . Need to set up DB name , DB password and APP_KEY.
10. To create tables in your database use this command - php artisan migrate .

What can you do with this project?

1. Login and Registartion 
2. View/Add/Edit/Delete - Sportnews Posts
3. Can set categories and tags for each Post
